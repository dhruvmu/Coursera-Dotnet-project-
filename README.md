# User Management API - TechHive Solutions

This is a complete ASP.NET Core Web API project for managing users in a corporate scenario.

## Features
- Full CRUD operations (GET, POST, PUT, DELETE)
- In-memory data storage (List<User>)
- Manual and automated input validation (Data Annotations)
- Global Error Handling Middleware
- Token-based Authentication Middleware
- Request Logging Middleware
- Swagger UI Integration

## Project Scenario
**TechHive Solutions** requires a robust API for managing their user base. This project demonstrates clean architecture patterns, enterprise-level middleware configuration, and reliable error handling.

## Getting Started
1. Clone the repository.
2. Ensure you have .NET 8/9 installed.
3. Run `dotnet run`.
4. Access Swagger UI at `http://localhost:5026/swagger`.
